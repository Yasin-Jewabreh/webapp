# 04: Gemeinsames Grundgerüst mit base.html und Bootstrap

## Meta

Status
: **Decided**

Updated
: 18-Jun-2026

## Problem Statement

Wie soll das gemeinsame Grundgerüst der Web-App aufgebaut werden, damit alle Seiten ein einheitliches Design haben?

## Decision

Wir nutzen eine base.html als gemeinsames Template für alle Seiten. Alle anderen Seiten erben davon mit Jinja2 extends. Bootstrap wird für das Styling genutzt.

*Decision was taken by:* github.com/MohamedAoun1

## Regarded Options

Wir haben zwei Optionen betrachtet:

+ Jede Seite hat eigenes HTML-Grundgerüst
+ Eine gemeinsame base.html für alle Seiten

| Criterion | Eigenes Grundgerüst | base.html |
| --- | --- | --- |
| **Wartbarkeit** | ❌ Änderungen müssen überall gemacht werden | ✔️ Einmal ändern, gilt für alle Seiten |
| **Konsistenz** | ❌ Navbar könnte auf jeder Seite anders aussehen | ✔️ Einheitliches Design auf allen Seiten |
| **Aufwand** | ❌ Mehr Code, mehr Fehlerquellen | ✔️ Weniger Code, zentralisiert |
